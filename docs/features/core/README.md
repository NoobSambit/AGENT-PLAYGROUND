## What Core Means

Core features are the base of the agent product:

- identity and persona
- chat turns and message persistence
- memory and recall
- emotional state and event history
- timeline aggregation
- learning state and adaptation
- personality evolution and deep profile state

If one of these changes, it usually affects multiple tabs, several routes, and at least one store or service boundary.
